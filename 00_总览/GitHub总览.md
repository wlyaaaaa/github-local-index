## 发布边界

- 私有备份仓库可按用户恢复需求保存敏感恢复材料；本公开索引只记录公开安全结论。
- 公开仓库在提交前执行暴露面审查，不记录 secret 值、原始日志、任务 XML 或私有 payload。
- Git 与 GitHub 事实由本仓库维护；机器路径、计划任务配置和恢复事实由 PCConfig 维护。
